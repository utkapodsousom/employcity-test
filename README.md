# Employcity Landing

Одностраничный лендинг, сверстанный на HTML/SCSS с использованием Gulp для сборки и деплоя на GitHub Pages.

## Скрипты
- `npm run dev` — сборка в dev-режиме с вотчером и browser-sync.
- `npm run build` — production-сборка в `dist/`.

## Сборка
```bash
npm ci
npm run build
```

## Деплой
Настроен GitHub Actions (`.github/workflows/deploy.yml`): при пуше в `main` выполняется `npm ci && npm run build`, артефакт `dist` деплоится на GitHub Pages.
